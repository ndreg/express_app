Hello 🖖, this is my first project using node and express. This part of the EDteam course **Node desde cero** that means **Node from zero**.

## Important stuff

**Development dependencies**

`> npm install pug -D`

`> npm install nodemon -D`

**Mainly dependencies**

`> npm install express`

**Execute using**

`> npm run server`
